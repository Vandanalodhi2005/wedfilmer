## 🎨 **Customization**

### Replace Logo:
1. Add your logo to `/public/logo.png`
2. Update Navbar.tsx line ~40
3. Update Footer.tsx line ~30
4. Change "WF" to your initials or logo image

### Change Colors:
Edit `app/globals.css`:
```css
@theme inline {
  --color-accent: #YOUR_COLOR;
  /* ...other colors */
}
```

### Update Images:
1. **Your own photos:** Add to `/public/images/`
2. **About page:** Edit `AboutSection.tsx` line 32
3. **Portfolio:** Edit `PortfolioSection.tsx` lines 15-28

### Update Content:
Edit `lib/constants.ts`:
- Services
- Testimonials
- Pricing packages
- FAQ items
- Contact info
